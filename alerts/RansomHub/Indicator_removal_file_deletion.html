Alert: File Deletion

Technique ID: T1070.004
Atomic Red Team – dostępne testy dla tej techniki:
  • Delete a single file - Windows cmd
    Polecenia testowe (cmd):
    del /f #{file_to_delete}
    Pobierz test_1.cmd
  • Delete an entire folder - Windows cmd
    Polecenia testowe (cmd):
    rmdir /s /q #{folder_to_delete}
    Pobierz test_1.cmd
  • Delete a single file - Windows PowerShell
    Polecenia testowe (powershell):
    Remove-Item -path #{file_to_delete}
    Pobierz test_1.ps1
  • Delete an entire folder - Windows PowerShell
    Polecenia testowe (powershell):
    Remove-Item -Path #{folder_to_delete} -Recurse
    Pobierz test_1.ps1
  • Delete Prefetch File
    Polecenia testowe (powershell):
    Remove-Item -Path (Join-Path "$Env:SystemRoot\prefetch\" (Get-ChildItem -Path "$Env:SystemRoot\prefetch\*.pf" -Name)[0])
    Pobierz test_1.ps1
  • Delete TeamViewer Log Files
    Polecenia testowe (powershell):
    New-Item -Path #{teamviewer_log_file} -Force | Out-Null
Remove-Item #{teamviewer_log_file} -Force -ErrorAction Ignore
    Pobierz test_1.ps1
  • Clears Recycle bin via rd
    Polecenia testowe (cmd):
    rd /s /q %systemdrive%\$RECYCLE.BIN
    Pobierz test_1.cmd

MITRE Description:
Adversaries may delete files left behind by the actions of their intrusion activity. Malware, tools, or other non-native files dropped or created on a system by an adversary (ex: [Ingress Tool Transfer](https://attack.mitre.org/techniques/T1105)) may leave traces to indicate to what was done within a network and how. Removal of these files can occur during an intrusion, or as part of a post-intrusion process to minimize the adversary's footprint. There are tools available from the host operating system to perform cleanup, but adversaries may use other tools as well.(Citation: Microsoft SDelete July 2016) Examples of built-in [Command and Scripting Interpreter](https://attack.mitre.org/techniques/T1059) functions include del on Windows, rm or unlink on Linux and macOS, and `rm` on ESXi.
